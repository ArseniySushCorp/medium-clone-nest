## Description

Medium clone on NestJS

## Installation

### Instal dependencies

```bash
$ yarn install
```

### Config db via .env

Remove .sample and add db config data

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## DB tools

```bash
$ yarn db:drop

$ yarn db:create

$ yarn db:migrate

$ yarn db:rollback
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
